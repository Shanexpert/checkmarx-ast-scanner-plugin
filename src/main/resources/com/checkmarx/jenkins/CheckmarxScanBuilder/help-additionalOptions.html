Additional options or parameters to be provided for CLI. For all the available options, refer the CLI documentation here
Few commonly used options -
Verbose Mode: -v, --verbose
Scan Preset: --sast-preset-name | e.g- --sast-preset-name "High and Medium"
Incremental SAST scan (Default false): --sast-incremental e.g- --sast-incremental false
Wait for scan completion (Default true) : -w, --nowait e.g --nowait false
